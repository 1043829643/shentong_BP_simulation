# DOTA2 BP 模拟工具 Demo

## 使用方式

1. 解压 `dota2-bp-demo.zip`。
2. 双击打开 `index.html`。
3. 在赛前配置页选择阵营、先后手和双方战队。
4. 点击“确定，进入 BP”后，在英雄池中单击英雄进行 Ban/Pick。
5. 在右侧 BP 方案时间轴中双击任意已选英雄，会从这一手开始取消后续所有选择。
6. Pick 后英雄默认出现在该方英雄池中最常使用的位置对应分路：1/5 号位为优势路，2 号位为中路，3/4 号位为劣势路。
7. 上方英雄栏会按阵营显示分路：天辉为“优势路 / 中路 / 劣势路”，夜魇为“劣势路 / 中路 / 优势路”。
8. 拖动上方英雄栏里的英雄头像到其他分路，可以改变当前方案的分路。
9. 点击“新方案”可复制当前 BP 进度和分路设置，并创建分支。
10. 点击“完成”进入总览页，对比所有方案的三路对线预测。

## 说明

- 这是纯前端 demo，不需要安装 Node、Python 或任何依赖。
- Demo 使用浏览器本地打开即可运行，不需要启动服务器。
- 英雄头像来自 Steam 官方 DOTA2 CDN；联网时会显示真实头像，离线时核心交互仍可使用。
- “其他英雄”使用 `7.41以来世界大赛_1-5号位简要英雄池.xlsx` 的 1-5 号位英雄池，按英雄名称排序，并自动排除我方英雄池中已出现的英雄。
- 头像右上角的红/绿三角是 demo 随机示意，不代表真实克制数据。
- Excel 上传入口已预留在界面中，当前 demo 不解析 Excel 内容。
